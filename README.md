## PROJETO COM REDUX E REDUX SAGA

Projeto configurado com Redux Wrapper e Redux-saga, excuta as sagas no servidor.

## Iniciando

Primeiro, rodar o servidor de desenvolvimento:

```bash
npm run dev
# or
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000)
